# CSS Variables Contract

**This is the locked vocabulary every Starting Point and base must use.** It is served at runtime by `GET /api/design-tokens.css` so a single branding change cascades to every app on the next page load — no rebuild required.

Every variable below is published on `:root`. Every consumer **must** include a fallback (`var(--color-primary, #0F766E)`) so an app degrades gracefully if the endpoint is briefly unreachable.

## Brand assets

| Variable | Default |
|---|---|
| `--logo-url` | empty string when no logo is set; otherwise `url("/brand-assets/<file>")` |
| `--logo-icon-url` | empty string when no icon is set; otherwise `url("/brand-assets/<file>")` |

Consumers must check for emptiness before applying the URL (e.g. `.logo[style*="--logo-url"] { background-image: var(--logo-url); }`).

## How to use

```css
/* Good: every var has a fallback */
.btn-primary {
  background: var(--color-primary, #0F766E);
  color: var(--color-bg, #FFFFFF);
  padding: var(--space-3, 0.75rem) var(--space-4, 1rem);
  border-radius: var(--radius-md, 0.5rem);
  font-family: var(--font-sans, system-ui, sans-serif);
  font-size: var(--text-base, 0.9375rem);
  box-shadow: var(--shadow-sm, 0 1px 2px rgba(15, 23, 42, 0.05));
}

/* Bad: no fallback */
.btn-primary { background: var(--color-primary); }

/* Bad: hex literal — breaks brand inheritance */
.btn-primary { background: #0F766E; }
```

## Where this is served

- Endpoint: `GET /api/design-tokens.css` (same-origin to every app)
- Generated by: `cortex/src/services/design-tokens-css.ts`
- Sources: base default tokens (`ekoa-data/bases/<base>/tokens.json`) overlaid with `company.branding.designSystem` from `~/.ekoa/data/company.json`
- Cache: ETag from `company.updatedAt`; `Cache-Control: max-age=60, must-revalidate`
- Linked from: `cortex/src/data/scaffold-templates/index.html` (every app)

## Migration note

Apps built before this contract existed continue to read `frontend/src/tokens.json` at build time and apply inline styles. They keep working but **do not auto-rebrand**. Rebuild to inherit the new pattern.
